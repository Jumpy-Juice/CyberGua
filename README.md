# CyberGua

## 我在做什么

我正在做一个把《周易》大衍筮法和当代网络文化结合的互动作品。  
这个项目面向线上访问场景，目标是让更多人用更直观、更现代的方式接触传统占筮文化。

我当前的主要工作包括：

- 持续打磨项目内容表达，明确作品定位与核心体验。
- 把经典文本整理为可长期使用的数字化资料，并用于作品呈现。
- 接入豆包相关能力，为作品提供 AI 参考解读能力。
- 优化线上可访问性和稳定性，让更多访问请求能够顺畅使用。
- 推进独立后端部署方案，为后续高访问量和数据能力做准备。

## 项目阶段

当前处于从“可演示原型”向“稳定线上版本”推进阶段。  
后续会继续围绕访问稳定性、内容质量与长期运营能力进行迭代。
